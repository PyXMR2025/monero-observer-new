The following feeds are currently available:

### Daily

- [/feed.xml](/feed.xml): default, includes *ALL stories/reports/messages/calendar/periodicals*.
	- [/feed-mini.xml](/feed-mini.xml): only fetches the most recent <50 items in *feed.xml*.
	- *(New)* [/feed-headlines.xml](/feed-headlines.xml): only includes the titles from *feed.xml*.
- [/feed-stories.xml](/feed-stories.xml): only includes *stories* and *periodicals* (no messages).
	- *(New)* [/feed-stories-mini.xml](/feed-stories-mini.xml): only fetches the most recent <50 items in *feed-stories.xml*.

### On demand

- [/feed-messages.xml](/feed-messages.xml): only includes on demand *MO Community messages*.
- [/feed-calendar.xml](/feed-calendar.xml): only includes upcoming *Calendar* events.

### Weekly

- [/feed-ct.xml](/feed-ct.xml): only includes the weekly *Cypherpunk Transmission* reports.
- *(New)* [/feed-dev.xml](/feed-dev.xml): only includes the weekly *Dev Activity Reports*.
- *(Inactive)* [/feed-ta.xml](/feed-ta.xml): only includes the weekly *MO XMR TA reports*.

### Monthly

- [/feed-blitz.xml](/feed-blitz.xml): only includes the monthly *MO Blitz reports*.

You can subscribe to any or all feeds.

*I have created a guide explaining how to [read Monero Observer inside your terminal using Newsboat](/read-monero-observer-terminal-newsboat/).*
