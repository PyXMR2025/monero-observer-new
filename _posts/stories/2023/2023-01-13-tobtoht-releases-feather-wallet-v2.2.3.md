tobtoht[^1] has released Feather Wallet[^2] version 2.2.3[^3]'[^4] with several bugfixes.

### Changes overview

- Fixed scaling issues on Windows and Linux
- Fixed an issue that caused Feather to be unable to connect to Trezor devices on macOS
- Minimum macOS version is now 10.14 (down from 10.15)
- (Re)-broadcasting transactions should no longer throw an error [..]


The complete list of improvements and fixes can be found on the Changelog[^5] page.

A GPG-signed list of the hashes[^6] is available on the official website and the signing key can be downloaded from multiple places[^7].

Feather is 100% community-sponsored and you can support development efforts by transferring any XMR amount to the project’s donation address[^8] or by accessing *Help → Donate to Feather* in the wallet.
