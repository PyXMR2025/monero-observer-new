rehrar[^1] has released Stack Wallet[^2] version 1.5.38[^3] with support for *MajesticBank*[^4] swaps and several other updates and improvements.

### Changes overview

```
Added MajesticBank for swapping 
Buy crypto with debit/credit card, powered by Simplex in-app
PayNyms for Bitcoin
New themes [..]
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^5] to learn more about the project.
