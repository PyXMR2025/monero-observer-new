rehrar[^1] has released Stack Wallet[^2] version 1.7.8[^3] with theme updates, a few fixes and improvements.

### Changes overview

```
Themes: Now only light/dark ship with the app, others downloadable
Added desktop custom block explorers
BCH removed wrapped segwit address type
Moved address book to isar
Epic Cash now fully functional
Firo fixes
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
