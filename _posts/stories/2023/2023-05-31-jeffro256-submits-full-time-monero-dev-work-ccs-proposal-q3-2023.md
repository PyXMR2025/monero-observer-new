```

ETA: ~4 months.
```

## Tasks overview

- Implementing Seraphis wallet code with Ukoe, jberman, et al.
- Drafting a formal specification for decoy selection and implementing it [..]

jeffro256 has contributed >=48 merged commits[^3] to the Monero core repository.
