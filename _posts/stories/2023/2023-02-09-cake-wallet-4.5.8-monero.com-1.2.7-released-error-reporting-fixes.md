Cake Labs[^1] has released *Cake Wallet* v4.5.8 and *Monero.com* v1.2.7[^2] with several error reporting fixes.

### Changes overview

- Reduce error reporting frequency
- Remove unrelated issues from error reports
- Fix errors reported in recent reports for stability
- Minor other fixes and code cleaning [..]

Consult the Github repository[^2] to verify and download the wallet APKs.

Note that Monero.com[^3] (iOS[^4], Android[^5]) is a Monero-only wallet and Cake Wallet[^6] (iOS[^7], Android[^8]) is a multi-coin wallet.
