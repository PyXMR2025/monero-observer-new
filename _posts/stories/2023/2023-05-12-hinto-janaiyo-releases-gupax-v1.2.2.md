hinto-janai[^1] has released *Gupax*[^2] version 1.2.2[^3] with several fixes, UI improvements and remote node changes.

### Changes overview

```
[Fixes]
- Fixed [Status] P2Pool stats overflowing sometimes
- Added help messages on config loading issues
- Fixed rare crash upon bad config data
[Updates]
- UI: Changed overall style (all text is monospace, darker theme, rounded corners)
- P2Pool: Color status is now ORANGE until synchronized
- XMRig: Color status is now ORANGE when not mining
- Remote Node changes [..]
```

The full changelog, sources, SHA256SUM and .asc files can be found on Github[^3] and on the *gupax.io* website[^4].

Consult hinto-janai's CCS proposal[^5] to learn more about the project.
