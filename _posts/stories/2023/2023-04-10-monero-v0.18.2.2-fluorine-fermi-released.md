Monero v0.18.2.2[^1] *Fluorine Fermi* is out[^2]'[^3] and binaries for both the official CLI[^4] and GUI[^5] wallets are now available for download:

> [CLI] This is the v0.18.2.2 release of the Monero software. This release adds a size limit for tx_extra in tx pool. [GUI] This release adds support for Ledger Stax.

### Release highlights

```
CLI:
- Daemon: add a size limit for `tx_extra` in tx pool
- Daemon: avoid log entries when re-blocking a subnet
- Daemon: fix deadlock and crash when syncing with full dataset on Window
- Daemon: don't drop peers for soft offenses
- Wallet: fix infinite loop in fake out selection
- Wallet: add Ledger Stax support
GUI:
- Add support for Ledger Stax
- Update p2pool to v3.2
- Minor bug fixes and UI improvements
```

To view the complete list of changes, compare v0.18.2.2 with the previous release (0.18.2.0) on Github[^6]. New binaries (CLI/GUI) can be found on the Downloads[^7] page on Getmonero.org.
