hinto-janaiyo[^1] has released version 1.2.0[^2] of the cross-platform GUI for mining Monero on P2Pool using XMRig, *Gupax*[^3].

### Changes overview

```
Added Benchmarks submenu in Status tab
Remote Node changes [..]
```

The full changelog, sources, SHA256SUM and .asc files can be found on Github[^2] and on the new *gupax.io* website[^4].

Consult hinto-janaiyo's CCS proposal[^5] to learn more about the project.

*Note: the maintainer has changed his Github username from 'hinto-janaiyo' to 'hinto-janai' and uploaded a new PGP key.*
