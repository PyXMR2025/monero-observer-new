Version 1.0.69[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with a hotfix and a few improvements.

### Changes overview

```
Hotfix: An info box has been added in case of being blocked by a firewall.
The wallet user interface has been improved.
The notification style has been enhanced.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
