Monero's *Fluorine Fermi Major Point Release 2*[^1] (v0.18.2.0) is out[^2]'[^3] and binaries for both the official CLI[^4] and GUI[^5] wallets are now available for download:

> [CLI] This release contains various bugfixes as well as improvements to block propagation time. [GUI] This release has a redesigned wallet creation UI and adds support for the upcoming P2Pool network upgrade.

## Highlights

- Daemon: add RingCT cache for huge block propagation speedup
- Daemon: refactored rx-slow-hash, add MONERO_RANDOMX_FULL_MEM env var to force use the full dataset for PoW verification
- Daemon: fix back ping to discover healthy peers to connect to
- Daemon: fix Dandelion++ fluff/stem bug with local transactions
- (CLI) Wallet: update hidapi/`libusb dependencies, fix Trezor connectivity issues on Windows
- (GUI) Update P2Pool to 3.0, add support for the upcoming P2Pool network upgrade
- (GUI) Redesign seed page, ask for seed confirmation on wallet creation
- (GUI) Android: disable local node usage [..]

To view the complete list of changes, compare v0.18.2.0 with the previous version (0.18.1.2) on Github[^6].

> This [CLI] release was the direct result of 11 people who worked, largely unpaid and altruistically, to put out 44 commits containing 865 new lines of code.

> This [GUI] release was the direct result of 9 people who worked, largely unpaid and altruistically, to put out 64 commits containing 1403 new lines of code.

Contributor lists for this release are available on Github[^7]'[^8].

New binaries (CLI/GUI) can be found on the Downloads[^9] page on Getmonero.org.

Before installing anything users should check that the file hashes match those on the signed list, and that the signature is valid[^10]. Guides for popular operating systems are available on Getmonero.org (Linux/Windows/Mac)[^11]'[^12] and Monero Observer (Linux)[^13]'[^14].

*Note that all P2Pool GUI miners need to upgrade to this version (0.18.2.0) to keep mining after March 18th[^15].*
