rehrar[^1] has released Stack Wallet[^2] version 1.7.10[^3], updating Monero to v0.18.2.2, after the recent decoy selection vulnerability disclosure[^4].

### Changes overview

```
Added ecash
Updated Monero to v0.18.2.2
Default to 12 word seeds for new wallets
Firo fixes
Themes v2. Now you can update them when we make changes
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^5] to learn more about the project.
