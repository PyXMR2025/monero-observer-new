rehrar[^1] has released Stack Wallet[^2] version 1.7.11[^3] with various fixes and improvements.

### Changes overview

```
Added custom PIN length
Changed how we handled connections to nodes under the hood (changed to sockets from pinging)
Namecoin send fix
Firo favorites card amount fix
Firo tx UI fixes
eCash fixes
Epic Cash tx status indicators
Litecoin ordinals scaffolding [..]
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^2].

Consult the previous Monero Observer report[^4] to learn more about the project.
