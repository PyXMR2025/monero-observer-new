Stack Wallet[^1] version 1.7.7[^2] was released with support for the privacy-focused exchange aggregator Trocador.app[^3], a bug fix and several improvements.

### Changes overview

```
Fix BCH pending send error
Added custom blockchains
Added option to randomize PIN pad 
Added Trocador as a swapping partner
Made swapping even smoother across the board
More UI/UX and animation enhancements [..]
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^2].

Consult the previous Monero Observer report[^4] to learn more about the project.
