Version 0.12.1[^1] of COMIT[^2]'s XMR-BTC atomic swap implementation (*xmr-btc-swap*[^3]) was released with the following change:

- Swap: merge separate cancel/refund commands into one `cancel-and-refund` command for stuck swaps

Downloads are available on the release page[^1].

Read the *Safety*[^4] section of the README.md file before using this tool and join COMIT's Matrix channel[^5] if you need help.

*Note that COMIT is no longer maintaing the project[^6]. binarybaron, lescuer97 and delta1 are the current maintainers of xmr-btc-swap.*
