ANONERO[^1] has released alpha version 0.4[^2] of *ANON*, the Android-only security and privacy-focused XMR app, with several new features.

### Changes overview 

```
- Encrypted backup (AES-256)
- Restore from encrypted backup
- Restore from seed (passphrase required)
- Manual wallet lock
- Secure wipe function
```

Note that the code repository, build instructions, and signed SHA256 hashes are also available on a new self-hosted Gitea hidden service[^3]. 

APK downloads can be found on the onionsite[^4], eepsite[^5] and in the associated Telegram channel[^6].

To support ANONERO's work, you can donate XMR to the address listed on the project's sites.

Consult the previous Monero Observer report[^7] to learn more about the project.
