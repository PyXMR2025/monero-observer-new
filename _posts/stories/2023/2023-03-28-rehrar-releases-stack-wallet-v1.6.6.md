rehrar[^1] has released Stack Wallet[^2] version 1.6.6[^3] with several fixes and improvements.

### Changes overview

```
Add 0-conf BCH
Fixed XMR/WOW wallet not syncing when launched straight into the wallet
Fixed tag address colors
Try https automatically for XMR/wow nodes
Swap send fixes
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
