pokkst[^1] has rebranded MyNero Wallet[^2] to *Mysu Wallet*[^3], with the release of version 0.4.2.1[^4].

### Changes overview

```
+ Added username and password option to node entries
+ Added Mysu (previously MyNero) Tor node as default node
+ Added multi-destination transactions, with a send screen redesign
* Node can now be changed during onboarding screen
* Rebranded to Mysu Wallet
```

To support the project you can donate XMR to the address listed on the website[^3].

*Note that this project's Github repository[^5] is no longer maintained[^6], as pokkst decided to move the source code to I2P[^7].*
