rehrar[^1] has released Stack Wallet[^2] version 1.7.1[^3] with several fixes and improvements.

### Changes overview

```
Add ETH and token send/receive functionality
Many UI fixes and updates
System brightness theme selection UI update
Firo private send and swap fix
Fix Wownero syncing
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
