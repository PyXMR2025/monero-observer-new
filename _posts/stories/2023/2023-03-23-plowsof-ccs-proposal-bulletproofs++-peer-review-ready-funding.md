plowsof[^1]'s CCS proposal[^2] to get the ePrint version[^3] of the Bulletproofs++[^4] paper[^5] *peer-reviewed* is now ready for funding[^6]:

```
Funding: 100% (130/130 XMR)
```

To support this proposal, you can donate some XMR to the address listed on its Gitlab *Funding Required*[^6] page.

To learn more about the project, consult the previous Monero Observer report[^7].

**Update 23/3/24: almost funded (~91% in 7 contributions).**

**Update 23/3/31: fully funded[^8].**
