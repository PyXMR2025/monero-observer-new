Version 1.0.67[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several fixes and improvements.

### Changes overview

```
Improved: Imperva DDoS protection now performs checks in the background.
Fix: API error messages have been translated for better understanding.
Fix: A cancel button has been added when opening a disputed trade from a push notification.
Enhanced: The PIN code can now be between 4 and 25 digits long [..]
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.

**Update: v1.0.68 was released with a hotfix (*Disable confusing popup about 503 errors during polling*)[^5].**
