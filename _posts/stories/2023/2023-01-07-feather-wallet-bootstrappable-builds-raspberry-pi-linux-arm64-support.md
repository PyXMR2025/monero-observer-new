Feather Wallet[^1] version 2.2.2[^2]'[^3] was released with major build system changes and support for Raspberry Pi and Linux ARM64:

> Feather Wallet releases are now bootstrappable. This means that we are able to build our entire toolchain and dependencies from their source code.

### Changes overview

- Build system changes (bootstrappable builds)[^4]
- Support for Raspberry Pi and Linux ARM64
- Qt was upgraded from 5.15 to 6.4
- macOS builds now ship as a .dmg file
- The webcam QR scanner now works on standalone Linux binaries
- Signed source tarballs can now be downloaded from the downloads page [..]

The complete list of improvements and bug fixes can be found on the *Changelog*[^5] page.

A GPG-signed list of the hashes[^6] is available on the official website and the signing key[^7] can be downloaded from the Github repository.
