Version 1.0.71[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with two fixes and an UI improvement.

### Changes overview

```
Hotfix: Fix the captcha on the login screen.
Hotfix: Register error with edited email.
Improved add edit screen.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
