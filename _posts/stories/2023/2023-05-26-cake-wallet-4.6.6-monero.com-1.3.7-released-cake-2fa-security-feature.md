Cake Labs[^1] has released *Cake Wallet* v4.6.6 and *Monero.com* v1.3.7[^2] with support for a new opt-in *nontraditional TOTP implementation* (*Cake 2FA*[^3]), an important update of the Monero code to v0.18.2.2, and several other bug fixes and improvements:

> We have released this version on all platforms except iOS, which is still waiting for review. We are expecting Apple to approve the release shortly.[^4]

### Changes overview

```
Opt-in to Cake 2FA for security
Auto generate restore height for Monero restore QR codes
Hausa and Yoruba languages
Additional privacy settings
Update Monero to 0.18.2.2
Refactoring and bug fixes
```

Consult the Github repository[^2] to verify and download the wallet APKs.

Note that Monero.com[^5] (iOS[^6], Android[^7]) is a Monero-only wallet and Cake Wallet[^8] (iOS[^9], Android[^10], Linux[^11], MacOS[^12]) is a multi-coin wallet.
