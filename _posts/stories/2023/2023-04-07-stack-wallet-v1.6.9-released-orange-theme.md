rehrar[^1] has released Stack Wallet[^2] version 1.6.9[^3] with a new orange theme, a few fixes and updates.

### Changes overview

```
Update Wownero to latest fork (doesn't sync yet)
Min confirmations of Epic to 3
Added orange theme
Fixed Epic Cash send all not calculating correctly
Added xpub viewing to Bitcoin derivative coins
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
