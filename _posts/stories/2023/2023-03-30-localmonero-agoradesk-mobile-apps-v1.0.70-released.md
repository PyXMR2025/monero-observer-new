Version 1.0.70[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with a bug fix and several improvements.

### Changes overview

```
Display all transactions of the wallet. Export them to CSV.
Fix: minimum feedback gets reset.
Display the distance to the local ad.
Display local ads in the user's country when local ads are selected.
All countries' names have been localized.
Added system theme switcher.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
