m2049r[^1] has released the latest version of the Monerujo[^2] wallet, v3.1.0 *Fluorine Fermi*[^3]:

> [UPDATE] Monerujo v3.1.0 "Fluorine Fermi" shows locked transactions & fixes sideshift timeouts

### Changes since 3.0.3

```
Mark locked transactions
Fix sideshift.ai integration
```

- All Monero Contributors
- @anhdres

Before usage, it is recommended to back up your seed and verify that you have downloaded the correct file using the SHA256 hashes[^3].

If you need help, consult the *Monerujo Quickstart Guides*[^4] and post your question on the Monerujo subreddit[^5].

To support the project, you can donate XMR to the address listed on Github[^6].
