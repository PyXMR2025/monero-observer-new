binarybaron[^1] has released version 0.4.0[^2] of *UnstoppableSwap*[^3], the XMR<>BTC atomic swaps GUI:

### Changes overview

```
Add labels to Monero and Bitcoin address textfields
Adjust the displayed minimum amount to account for network fees
Upgrade tor version to 12.0.1
Allow manual cancel & refund 
Calculator for estimating amount of XMR user will receive
Remaining funds alert 
Auto Updater
```

Consult the previous Monero Observer report[^4] to learn more about the associated CCS proposal[^5].
