selsta[^1] has posted a monthly CCS progress report[^2] for March 2023, which includes the following dev updates:

```
Milestone 2:
- Prepared v0.18.2.1 and v0.18.2.2 (compiling and packaging binaries, release notes, blog post)
- Added support for Ledger Stax
- Research and block sybil nodes via DNS: https://b10c.me/observations/06-linkinglion/
- Smaller fixes, improvements and maintenance work (CLI, GUI)
```

Note that *misc* work is not explicitly mentioned in these updates. The full list of changes can be found on Github[^3]'[^4].

This update completes the second milestone of selsta's part-time (Feb-April) Monero development CCS proposal[^5].
