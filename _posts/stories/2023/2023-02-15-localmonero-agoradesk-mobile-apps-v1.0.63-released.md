Version 1.0.63[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with the following fixes:

```
Fixed update for push notification token.
Fixed initial screen for the affiliate program.
Made numerous small fixes.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
