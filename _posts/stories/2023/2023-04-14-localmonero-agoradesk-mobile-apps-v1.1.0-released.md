Version 1.1.0[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with a  few improvements and a bug fix.

### Changes overview

```
The pin code is limited to 25 attempts; after that, the app will log out.
Pressing the proxy icon opens the settings screen.
The transaction cost for BTC has been added in fiat currency.
Fixed: error that occurred when the captcha dialog was closed.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
