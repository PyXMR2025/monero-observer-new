Version 1.0.65[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several fixes and improvements.

### Changes overview

```
Hotfix: Addressed captcha loop during service attack
Fixed: Wallet displaying zero balance
Fixed: Calculated price update in ad formula when switching
Added: Settlement proof button to completed trades
Added: Proxy setting before login
Enhanced: Wallet tiles can be minimized for compact view
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
