tobtoht[^1] has released Feather Wallet[^2] version 2.4.3[^3]'[^4] with several improvements and fixes:

```
- AppImages for Tails / Whonix / Prestium no longer bundle a Tor binary ..
- the built-in updater should now work on macOS
- macOS builds are once again shipped as application bundles, instead of .DMG files
- Receive: added a column that shows the subaddress index
- new wallets should now always start syncing from the current blockheight
- Fixed a build issue on Qt 5 [..]
```

The complete list of changes can be found on the Changelog[^5] page.

A GPG-signed list of the hashes[^6] is available on the official website and the release signing key can be downloaded from multiple locations[^7].

Feather is 100% community-sponsored and you can support development efforts by transferring any XMR amount to the project’s donation address[^8] or by accessing *Help → Donate to Feather* in the wallet.
