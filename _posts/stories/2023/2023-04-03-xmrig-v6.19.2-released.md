*XMRig*[^1] version 6.19.2[^2] has been released with several fixes and improvements.

### Changes overview

```
#3230 Fixed parsing of TX_EXTRA_MERGE_MINING_TAG.
#3232 Added new X-Hash-Difficulty HTTP header.
#3240 Improved .cmd files when run by shortcuts on another drive.
#3241 Added view tag calculation (fixes Wownero solo mining issue).
```

Verify the SHA256 sums with xmrig[^3]'s GPG key[^4] (ID: 446A53638BE94409) before using the software.

Consult the README.md[^5] file for more information about XMRig and transfer Monero to the project's donation address[^6] if you want to support the project.
