tobtoht[^1] has released Feather Wallet[^2] version 2.3.0[^3]'[^4] with several improvements and fixes.

### Changes overview

- verify tx proof: added option to verify a payment using the transaction secret key
- wizard: added option to set subaddress lookahead when restoring from device, seed, keys
- wizard: added option to override embedded wallet creation date for 14/16-word seeds
- wizard: allow restoring deterministic wallet from private spend key [..]

The complete list of changes can be found on the Changelog[^5] page.

A GPG-signed list of the hashes[^6] is available on the official website and the release signing key can be downloaded from multiple locations[^7].

Feather is 100% community-sponsored and you can support development efforts by transferring any XMR amount to the project’s donation address[^8] or by accessing *Help → Donate to Feather* in the wallet.
