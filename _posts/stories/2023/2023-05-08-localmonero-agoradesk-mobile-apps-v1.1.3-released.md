Version 1.1.3[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with a few fixes and new icons for payment providers.

### Changes overview

```
Fix: Removed redundant maintenance mode dialog popup.
Feature: Introduced new icons for various payment providers.
Fix: Push notification tokens now refresh every 15 days.
Fix: Addressed null error issue when an SVG file is absent.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
