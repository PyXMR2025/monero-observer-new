rehrar[^1] has released Stack Wallet[^2] version 1.6.4[^3] and announced the first version of *Stack Duo*[^4], a Bitcoin/Monero only wallet:

> It's not perfect. Some illustrations need updating. Might be a bug here or there. But it's ready to go. And we couldn't be more thrilled to welcome Stack Duo to our Stack suite.[^5]

### Changes overview

```
Higher quality chans
Firo fixes
Address details screens
Stacy Stack chan loading screen asset update 
Chans in dark theme [..]
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^6] to learn more about the project.
