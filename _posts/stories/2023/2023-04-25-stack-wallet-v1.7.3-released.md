rehrar[^1] has released Stack Wallet[^2] version 1.7.3[^3] with several fixes and improvements.

### Changes overview

```
Bitcoin and BCH remove derivation paths stored alongside addresses.
Fix BCH 0-conf bugs
Updated animations for sending, syncing, and more
Set WOW confs to 15 before spending
Updated default block explorer for Bitcoin to mempool.space
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
