busyboredom[^1]'s CCS proposal[^2] to create a *standalone, dockerized, AcceptXMR[^3]-based payment gateway with wordpress support* is now ready for funding[^4]:

```
Total funding needed: 14 XMR
```

To support this proposal, you can donate some XMR to the address listed on its Gitlab *Funding Required*[^4] page.

To learn more about the project, consult the previous Monero Observer report[^5].

**Update: fully funded[^6].**
