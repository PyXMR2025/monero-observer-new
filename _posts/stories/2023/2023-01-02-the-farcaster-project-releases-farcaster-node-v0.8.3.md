The Farcaster Project[^1] has released version 0.8.3[^2] of the Farcaster cross-chain atomic swap node[^3]:

### Changes overview

    Swapd: Don't add color coding to progress message
    Farcasterd: Add delay between xmr auto-funding reattempts
    Swapd: Simplify key manager encoding
    Swapd: Simplify commitment creation
    [..]

To learn more about the project, consult the previous Monero Observer report[^4].

*Note that although a mainnet release is very close[^5], the tech is still experimental.*
