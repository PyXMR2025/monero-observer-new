busyboredom[^1] has submitted a CCS proposal (draft)[^2] looking to create a *standalone, dockerized, AcceptXMR[^3]-based payment gateway with wordpress support*:

> This proposal aims to make AcceptXMR usable for anyone capable running a docker container and
installing a wordpress plugin.

```
Total funding: 30 XMR.

ETA: TBD.
```

### Milestones overview

- M1: Dockerize AcceptXMR (14 XMR)
- M2: Wordpress Plugin (10 XMR)
- M3: Maintenance (until 2024) (6 XMR)

CCS coordinator plowsof suggested funding this proposal *entirely by the remaining funds of xmrSale*[^4].

More information about busyboredom's work is available on *busyboredom.com*[^5].

To read the full proposal, share your feedback, ask questions and support this CCS, consult !374[^2].

*This is an ongoing story and the report will be updated when new information is available.*
