Stack Wallet[^1] version 1.6.1[^2] was released with a new *coin control* feature for mobile and several UI fixes.

### Changes overview

```
Added coin control for mobile (Linux soon)
Many UI updates and fixes
Changed Exchange to Swap
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^2].

Consult the previous Monero Observer report[^3] to learn more about the project.
