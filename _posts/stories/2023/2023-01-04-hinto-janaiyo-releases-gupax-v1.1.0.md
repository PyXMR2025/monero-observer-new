hinto-janaiyo[^1] has released version 1.1.0[^2] of the cross-platform GUI for mining Monero on P2Pool using XMRig, *Gupax*[^3].

### Changes overview

- Status: Added P2Pool submenu
- Status: Added more process stats 
- macOS: Added warning (and solution) if Gupax/P2Pool/XMRig were quarantined by Gatekeeper
- P2Pool/XMRig: Fixed processes sometimes not starting after entering a custom PATH [..]

The full changelog, sources, SHA256SUM and .asc files can be found on Github[^2] and on the new *gupax.io* website[^4].

Consult hinto-janaiyo's CCS proposal[^5] to learn more about the project.
