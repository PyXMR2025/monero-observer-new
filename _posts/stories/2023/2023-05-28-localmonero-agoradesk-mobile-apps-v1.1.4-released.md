Version 1.1.4[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with two fixes and one UI improvement.

### Changes overview

```
Added action buttons for disputed trades within the trade chat feature.
Fixed: Ensured compatibility of SOCKS5 with Orbot.
Fixed: Implemented a check for fields with commas during the ad creation process.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
