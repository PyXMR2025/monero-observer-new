m2049r[^1] has announced[^2] a new Monerujo[^3] minor release, version 3.2.0 *Decoy Selection*[^4], which upgrades to Monero Core 0.18.2.2:

> Monerujo v3.2.0 "Decoy Selection" has been released! [..] Shout outs to all contributors & supporters!!

### Changes since 3.1.3

```
Upgrade to Monero Core v0.18.2.2
Improve Restore Height Calculations
```

### Contributors

- All Monero Contributors
- @anhdres
- @Baltsar
- @m2049r


Before usage, it is recommended to back up your seed and verify that you have downloaded the correct file using the SHA256 hashes[^3].

If you need help, consult the *Monerujo Quickstart Guides*[^5] and post your question on the Monerujo subreddit[^6].

To support the project, you can donate XMR to the address listed on Github[^7].
