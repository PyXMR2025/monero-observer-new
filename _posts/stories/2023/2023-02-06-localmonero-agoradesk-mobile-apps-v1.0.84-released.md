Version 1.0.84[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with the following fixes and improvements:

- Improved login with Imperva captcha.
- Fixed creating ad issues.

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
