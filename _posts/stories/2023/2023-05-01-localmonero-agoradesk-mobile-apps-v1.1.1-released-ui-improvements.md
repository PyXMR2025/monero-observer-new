Version 1.1.1[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several UI improvements.

### Changes overview

```
Enhancement: Improved the display of the maintenance mode dialog.
Refined dropdown list styles and icons.
Enhanced profile screen styles.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
