The Farcaster Project[^1] has released version 0.8.4[^2] of the Farcaster cross-chain atomic swap node[^3]:

### Changes overview

    Swapd: Refactor key manager init 
    Fix: filter by network all deal list responses
    Fix(grpc): list Monero funding addresses
    Fix(state report): accordant lock used arbitrating params 
    Syncer: Bump Monero to ring size 16
    Fix(tsm): swaps restoring

To learn more about the project, consult the previous Monero Observer report[^4].

*Note that although a mainnet release is very close[^5], the tech is still experimental.*
