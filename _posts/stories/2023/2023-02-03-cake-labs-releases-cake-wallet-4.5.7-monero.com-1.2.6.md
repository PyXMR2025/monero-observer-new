Cake Labs[^1] has released *Cake Wallet* v4.5.7 and *Monero.com* v1.2.6[^2] with several improvements and fixes.

### Changes overview

- You can now grab BTC/LTC/XMR address from Twitter bios!
- QR code fixes and enhancements
- Add error reporting option
- Add Burmese, Turkish, Thai, and Arabic [..]

Consult the Github repository[^2] to verify and download the wallet APKs.

Note that Monero.com[^3] (iOS[^4], Android[^5]) is a Monero-only wallet and Cake Wallet[^6] (iOS[^7], Android[^8]) is a multi-coin wallet.
