selsta[^1] has posted a monthly CCS progress report[^2] for February 2023, which includes the following dev updates:

```
Milestone 1:
- v0.18.2.0 released (compiling and packaging binaries, release notes, blog post)
- Hardware wallet testing after dependencies got updated
- Continued with issue tracker cleanup
- Smaller fixes, improvements and maintenance work (CLI, GUI)
```

Note that *misc* work is not explicitly mentioned in these updates. The full list of changes can be found on Github[^3]'[^4].

This update completes the first milestone of selsta's part-time (Feb-April) Monero development CCS proposal[^5].
