selsta[^1] has posted a monthly CCS progress report[^2] for January 2023, which includes the following dev updates:

```
Milestone 3:
- v0.18.2.0 is now ready and will be released in the coming days
- Bring GUI dependencies up to date (hidapi, Qt)
- More seed node maintenance that is now merged, multiple offline seed nodes have been removed and new ones added
```

Note that *misc* work is not explicitly mentioned in these updates. The full list of changes can be found on Github[^3]'[^4].

This update completes the third and final milestone of selsta's part-time (Nov-Jan) Monero development CCS proposal[^5].
