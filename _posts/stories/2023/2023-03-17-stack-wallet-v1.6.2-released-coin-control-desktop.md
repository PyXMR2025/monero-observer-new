Stack Wallet[^1] version 1.6.2[^2] was released and it expands support for their new *coin control*[^3] feature to desktop.

### Changes overview

```
- Added coin control to desktop
- Added coin control features and fixes
 * Sort by address, age, or output
```

The release notes, binaries, and the SHA256 hashes can be found on Github[^2].

Consult the previous Monero Observer report[^4] to learn more about the project.
