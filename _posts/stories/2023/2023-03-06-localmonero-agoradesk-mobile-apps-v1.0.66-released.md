Version 1.0.66[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several fixes.

### Changes overview

```
Fixed: Local trades funded and other bugs.
Fixed: Asseccibility search fields in dropdown lists.
Fixed: The app doesn't show progression to step 2 of XMR settlement.
Fixed: minimum feedback requirement in the app is not saved with the rest of the ad.
```

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
