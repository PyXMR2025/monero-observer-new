CryptoGrampy[^1] has released version 0.4.0-alpha[^2] of *xmr-pocket-node*[^3], a simple full/pruned Monero GUI node for Android:

> Updated the #Monero PocketNode binaries/apk-release to v0.18.2.0 for anyone interested[^4]

### Changes overview

```
Updates armv7/v8 binaries and APK to use Monero v0.18.2.0.
```

To support CryptoGrampy's work, you can donate XMR to the address listed at the bottom of the  *README.me*[^5] file.

CryptoGrampy is still[^6] looking[^7] for experienced Android developers to maintain and contribute to the project.

*Note that this project is still a WiP. Use at your own risk.*
