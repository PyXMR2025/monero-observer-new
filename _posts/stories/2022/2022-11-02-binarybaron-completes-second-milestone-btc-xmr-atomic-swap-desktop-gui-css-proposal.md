binarybaron[^1] has completed[^2] the second milestone for their XMR-BTC atomic swap desktop GUI prototype CCS proposal[^3]:

> I have resumed my work on the project and have been putting in extra effort to catch up.
I will still complete everything included in the proposal for the same amount of community resources, it will just take me a little longer than I had originally anticipated.

### Work overview

- GUI: Adjust the displayed minimum amount to account for network fees[^4]
- GUI: Upgrade tor version to 11.5.4[^5]
- GUI: Improved transaction monitoring to allow for instantous detection of possible refund scenario[^6]
- xmr-btc-swap: Coordinated with the other maintainers to release version 0.11.1[^7]
- xmr-btc-swap: Started working on specification and planning of API for xmr-btc-swap cli and set bounty of 3 XMR for the first person to implement it[^8]
- API: had an uptime of 100% last month.[^9]
- Rendezvous Point Maintenance[^10]
- Maintenance of the front end[^11]
- [..] 

Read the full update[^2] on Github for more details.

Consult the previous Monero Observer report[^12] to learn more about the project.
