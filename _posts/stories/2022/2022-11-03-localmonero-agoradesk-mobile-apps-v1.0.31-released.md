Version 1.0.31[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several fixes and updates:

- Feat: if user dismiss push message he will be marked as online.
- Feat: FOSS versions - alert dialog when new version is available.
- Feat: tooltips for ad actions.
- Fix: accessibility improvements - named buttons and elements.
- Fix: managing badge on the app icon when push messages have tapped.

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
