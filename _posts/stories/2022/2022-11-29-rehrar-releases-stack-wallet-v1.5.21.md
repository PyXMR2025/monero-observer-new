rehrar[^1] has released Stack Wallet[^2] alpha version 1.5.21[^3] with several fixes and improvements.

### Changes overview

- [General] Added emoji search
- [General] Allow node deletion if at least one node still exists
- [Desktop] Several UI fixes, updates, optimizations
- [Desktop] Added animations [..]

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
