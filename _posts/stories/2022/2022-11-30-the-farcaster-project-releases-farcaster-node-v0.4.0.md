The Farcaster Project[^1] has released version 0.4.0[^2] of the Farcaster cross-chain atomic swap node[^3]:

### Changes overview

    doc: move documentation to wiki
    Add Support for electrum client Tor proxy
    swapd: improve PendingRequests and add associated fn defer_request
    fix(nix): fix git revision and format file according to statix
    ci: remove nightly usage
    [..]

To learn more about the project, consult the previous Monero Observer report[^4].

*Note that the tech is still experimental, use it on mainnet at your own risk!*
