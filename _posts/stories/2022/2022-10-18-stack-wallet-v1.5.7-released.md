Stack Wallet[^1] beta version 1.5.7 (build_0077)[^2] has been released with several bug fixes and improvements:

- Added Incognito mode
- Fix some issues with SWB and Log savings
- Firo now works!
- Added SimpleSwap
- House keeping fixes on confirmations to unlock and decimal places
- Fixed app crashes related to Wownero 
- Updated exchange flow [..]	

The release notes for this version, the .APK download and the SHA256 hashes can be found on Github[^2]. 

Consult the previous Monero Observer report[^3] to learn more about the project.
