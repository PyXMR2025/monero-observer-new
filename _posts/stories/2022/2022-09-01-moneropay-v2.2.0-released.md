Version 2.2.0[^1] of the *MoneroPay*[^2] standalone XMR payment gateway was released.

There are no release notes for this version.

To see the software in action, visit donate.kernal.eu[^3].

Patches can be sent via email[^4] and dev discussions take place on Matrix[^5].
 
Consult the MoneroPay documentation[^6] for deployment instructions and API specs.
