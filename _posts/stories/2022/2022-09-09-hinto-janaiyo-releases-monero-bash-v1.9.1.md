hinto-janaiyo[^1] has released version 1.9.1[^2] of the Monero Bash wrapper for Linux, *monero-bash*[^3].

## Updates and fixes

- HTTP Spoofing: Added TOR_BROWSER_MIMIC option that mimics the HTTP headers used by Tor browser
- HTTP Spoofing: Added ONLY_WGET_CURL option to only use random wget/curl versions as the User-Agent
- Watch: Spacing is auto-adjusted on terminal size changes
- Status: P2Pool stat calculation is slightly faster
- [..]

The full list of changes and the sources, SHA256SUM and .asc files can be found on Github[^2]. Always verify before using.

To learn more and support the project, consult hinto-janaiyo's new CCS proposal[^4] to finish developing monero-bash v2.
