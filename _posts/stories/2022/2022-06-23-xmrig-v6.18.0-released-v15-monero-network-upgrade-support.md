*XMRig*[^1] version 6.18.0[^2] has been released and it adds support for the upcoming v15 Monero network upgrade[^3]:

> Monero v15 network upgrade support and more house keeping.[^4]

### Other changes

  * Fixes for 32-bit ARM.[^5]
  * Fixed being unable to resume from `pause-on-battery`.[^6]
  * Fixed `--cpu-priority` not working sometimes.[^7]
  * Removed old AstroBWT algorithm.[^8]

Verify the SHA256 sums with xmrig[^9]'s GPG key[^10] (ID: 446A53638BE94409) before using the software.

Consult the README.md[^11] file for more information about XMRig and transfer Monero to the project's donation address[^12] if you want to support the project.
