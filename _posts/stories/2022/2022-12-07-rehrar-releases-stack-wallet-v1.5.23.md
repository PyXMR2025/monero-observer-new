rehrar[^1] has released Stack Wallet[^2] alpha version 1.5.23[^3] with several fixes and improvements.

### Changes overview

- [General] Further updates to recovery phrase explanation
- [General] Bug fixes and optimizations
- [Desktop] Several UI fixes, updates, optimizations
- [Desktop] Fixed bug where sometimes wallets don't show up
- [Desktop] Fixed theme updating bugs

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
