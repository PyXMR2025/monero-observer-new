ANONERO[^1] has released alpha version 0.2.1[^2] of *ANON*, the Android-only security and privacy-focused XMR app. 

This release *enforces proxy connection with default 9050/tor, and adds onion node support*:

```
- enforce proxy
- onion node support
```

APK downloads are also available on the onionsite[^3], eepsite[^4] and in the associated Telegram channel[^5]. Build instructions and signed SHA256 hashes can be found on Gitea[^6].

To support ANONERO's work, you can donate XMR to the address listed on the project's sites.

Consult the previous Monero Observer report[^7] to learn more about the project.
