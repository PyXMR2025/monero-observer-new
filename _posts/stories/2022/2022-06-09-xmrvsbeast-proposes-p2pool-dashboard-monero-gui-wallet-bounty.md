xmrvsbeast[^1] has proposed a bounty[^2] for adding a P2Pool[^3] statistical mining dashboard to the Monero GUI Wallet[^4]:

> This bounty is for a graphical mining dashboard for p2pool in the Monero GUI wallet that displays and graphs metrics over time where appropriate

```
Total Bounty: 0.089609691025 XMR (to date)
```

### Requirements

- Code is open-source
- Code is submitted to the Monero repo as a pull-request[^5]
- Pull request is accepted into Monero's code base after review

To start working on the project yourself, you should make your intentions public by posting a comment in the bounty's thread.

To increase the bounty you can contribute some XMR to the bounty address posted by the *Monero Bounties Bot*[^2].

Discuss the proposal in the associated Reddit thread[^6].

*This is an ongoing story and the report will be updated if/when new information is available.*
