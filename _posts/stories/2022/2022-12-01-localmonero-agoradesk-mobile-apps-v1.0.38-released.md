Version 1.0.38[^1] of the *LocalMonero/AgoraDesk*[^2]'[^3] mobile apps has been released with several fixes and support for new languages:

- Feat: added a lot of new languages.
- Fix: improved brute-force attack captcha.
- Fix: Android: able to open trade chat without PIN.
- Fix: fixed few accessibility bugs.
- Fix: account created times are different with web.
- Fix: view receipt bug.
- Fix: tradeview fixes for deleted accounts.

The full list of changes and the APK downloads are available on Github[^1].

Consult the previous Monero Observer report[^4] to learn more about the apps.
