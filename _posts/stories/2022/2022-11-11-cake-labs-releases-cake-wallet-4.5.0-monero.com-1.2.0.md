Cake Labs[^1] has released *Cake Wallet* v4.5.0 and *Monero.com* v1.2.0[^2] with several fixes and improvements:

- Flutter update
- Exchange improvements
- Buy functionality update with Onramper
- Reinstatement of improved backup functionality using new Flutter features
- Small UI, bug fixes

Consult the Github repository[^2] to verify and download the wallet APKs.

Note that Monero.com[^3] (iOS[^4], Android[^5]) is a Monero-only wallet and Cake Wallet[^6] (iOS[^7], Android[^8]) is a multi-coin wallet.
