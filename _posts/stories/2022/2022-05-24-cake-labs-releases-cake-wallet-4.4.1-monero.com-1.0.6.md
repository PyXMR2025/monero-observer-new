Cake Labs[^1] has released *Cake Wallet* v4.4.1 and *Monero.com* v1.0.6[^2]:

> Cake Wallet 4.4.1 and Monero.com 1.0.6 are both live on Google Play, GitHub APK, and the App Store. This release features SideShift and many quality of life improvements :)[^3]

### Changes

    Added SideShift exchange for fixed and variable rates
    Allowing saving templates with the fiat amount
    Changed default node for Haven (Cake Wallet only)
    Translation fixes
    Yat fixes
    Show subaddress label on the transactions details page
    Hidden balance mode fixes
    Increase tapping area on the dashboard screen
    Collapse button for address book
    Bug fixes and other minor fixes

    
Consult the Github repository for the full list of changes[^2].

Monero.com[^4] is a Monero-only wallet and Cake Wallet[^5] is a multi-coin wallet. Read my previous report to learn more[^6].
