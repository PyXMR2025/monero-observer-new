selsta[^1] has posted a monthly CCS progress report[^2] for August 2022, which includes the following dev updates:

```
Milestone 1:
- Skip bootstrap nodes that are lower than last checkpoint
- Preparing a new release to fix some bugs introduced with v0.18 (v0.18.1.1)
- lmdb update to master branch testing
- Investigate slow daemon exit on macOS in some cases, add a workaround for GUI simple mode
- Smaller fixes and improvements (CLI, GUI)
```

*Misc* work is not explicitly mentioned in these updates.

A more complete list of changes can be found on Github[^3]'[^4].

Consult the previous Monero Observer report[^5] to learn more about selsta's most recent CCS proposal.
