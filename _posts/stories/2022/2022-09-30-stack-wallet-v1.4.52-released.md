Stack Wallet[^1] beta version 1.4.52 (build_0068)[^2] has been released with support for Wownero, Bitcoin Cash, and Namecoin:

- Namecoin added
- Bitcoin Cash added
- Wownero added
- Confirms null check
- Confirmations fix

The release notes for this version, the .APK download and the SHA256 hashes can be found on Github[^2]. 

Consult the recent Monero Observer report[^3] to learn more about the project.
