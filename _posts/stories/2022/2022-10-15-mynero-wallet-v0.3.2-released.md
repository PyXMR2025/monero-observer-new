pokkst[^1] has released versions 0.3.0, 0.3.1, and 0.3.2[^2] of MyNero Wallet[^3] with several improvements and bugfixes:

```
- Adds feature to edit/delete nodes in node list [0.3.0]
- Adds "churning" option when selecting UTXOs [..] [0.3.0]
- Fixes crashing issue with users updating who had not set a node manually [0.3.1]
- Fixes issue with node preference upgrade process [0.3.2]
```

The complete list of changes is available on Github[^4].

A GPG-signed list of the hashes[^2] and the signing key[^5] can be downloaded from the project's website.

To support the project you can donate XMR to the address listed on the website and in the repository README.md file[^7].

*Note: MyNero was only announced a few weeks ago[^6] and is still in early development.*
