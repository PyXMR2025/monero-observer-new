Stack Wallet[^1] beta version 1.5.14 (build_0086)[^2] has been released with the following bug fixes and improvements:

- Fixed TX Details screen
- Fixed search not working on All Transactions view
- Many more desktop screens

The release notes for this version, the .APK download and the SHA256 hashes can be found on Github[^2]. 

Consult the previous Monero Observer report[^3] to learn more about the project.
