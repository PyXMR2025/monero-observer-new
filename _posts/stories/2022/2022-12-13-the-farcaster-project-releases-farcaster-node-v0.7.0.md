The Farcaster Project[^1] has released version 0.7.0[^2] of the Farcaster cross-chain atomic swap node[^3]:

### Changes overview

    Swap wallet
    Config: expose swap finality parameters
    Tests: Add retries to grpc progress message test
    Grpc: Handle recoverable server runtime panics
    Feat: Use creation height when sweeping Monero
    [..]

To learn more about the project, consult the previous Monero Observer report[^4].

*Note that although a mainnet release is very close[^5], the tech is still experimental.*
