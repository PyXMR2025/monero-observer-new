DangerousFreedom1984[^1]'s CCS proposal[^2] to develop a Seraphis PoC CLI wallet and audit framework is now ready for funding[^3]:

```
28 contributors raised 129.04 of 120 XMR (~107%)
```

To support this proposal, you can donate XMR to the address listed on its Gitlab *Funding Required*[^3] page.

To learn more about the project, consult the previous Monero Observer report[^4].
