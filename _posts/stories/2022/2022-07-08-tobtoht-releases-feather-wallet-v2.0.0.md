tobtoht[^1] has released Feather Wallet[^2] version 2.0.0[^3]:

> This release is ready for the August hardfork. [..] also added support for the Ledger Nano S+.[^4]

### Features

```
- New Polyseed seed scheme
- Coin control: manual input selection
- Settings: new Privacy tab
- Home Tab additions
```

The complete list of improvements and bug fixes can be found in the release announcement[^4].

A GPG-signed list of the hashes[^5] is available on the official website and the signing key[^6] can be downloaded from the Github repository.
