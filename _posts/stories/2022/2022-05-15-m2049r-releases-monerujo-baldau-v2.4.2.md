m2049r[^1] has released the latest version of the Monerujo[^2] wallet, v2.4.2 *Balda큼*[^3]:

> This is minor release v2.4.2 "Balda큼" upgrading to Monero Core v0.17.3.2 & adding support for Ledger Nano S plus

### Changes since 2.4.1

```
- Upgrading to Monero Core v0.17.3.2
- Add support for Ledger Nano S plus
- Updated Russian & Brazilian Translations
- Fix crashing when adding/testing node
```
