vdo[^1] has released v0.4.0[^2] of the Monero node boostrapper script, *xmr.sh*[^3]:

### Changes

```
Added P2Pool service
Monero-LWS updated to v0.2_0.18
Tor updated to v0.4.7.8
```

To learn more about the project, consult the project's code repository[^6], the FAQ[^7] section, and the asciinema.org demo[^8].
