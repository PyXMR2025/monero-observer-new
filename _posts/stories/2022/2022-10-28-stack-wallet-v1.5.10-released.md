Stack Wallet[^1] beta version 1.5.10 (build_0082)[^2] has been released with several bug fixes and improvements:

- Fixed Stack Wallet Backup and logs saving for iOS and Android <10
- Added new copy feature for easy bug reporting
- Added support for new Firo nodes
- Bug fixes, optimizations
- More desktop UI work

The release notes for this version, the .APK download and the SHA256 hashes can be found on Github[^2]. 

Consult the previous Monero Observer report[^3] to learn more about the project.
