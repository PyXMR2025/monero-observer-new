pokkst[^1] has released version 0.1.0[^2] of MyNero Wallet[^3], a Monerujo fork:

> MyNero Wallet 0.1.0 is now available! A no-bullshit, pure Monero wallet suitable for both newcomers and experienced users.[^4]

### Changes overview

```
- Add toast when selecting node
- Fee selection and code cleanup
- Change license [..]
```

The complete list of improvements and bug fixes can be found on Github[^5].

A GPG-signed list of the hashes[^2] and the signing key[^6] can be downloaded from the project's website.

*Note: MyNero was only announced a few weeks ago[^7] and is still in early development.*
