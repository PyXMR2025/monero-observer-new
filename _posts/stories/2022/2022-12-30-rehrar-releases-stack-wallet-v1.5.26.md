rehrar[^1] has released Stack Wallet[^2] alpha version 1.5.26[^3] with several fixes and improvements.

### Changes overview

- Monero syncing fixes
- Exchanges page optimizations
- Logging sanitize and fixes
- Firo fixes and optimizations (faster syncs and spends)
- Epic node addition fixes and fee selection fix

The release notes, binaries, and the SHA256 hashes can be found on Github[^3].

Consult the previous Monero Observer report[^4] to learn more about the project.
