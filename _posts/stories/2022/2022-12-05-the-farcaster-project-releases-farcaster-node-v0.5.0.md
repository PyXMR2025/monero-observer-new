The Farcaster Project[^1] has released version 0.5.0[^2] of the Farcaster cross-chain atomic swap node[^3]:

### Changes overview

    Refactor: Improve reliability of peerd connections
    Add richer data to the progress messages
    Swapd: Simplify checkpoint encoding
    Swapd: Some patches and improvements
    [..]

To learn more about the project, consult the previous Monero Observer report[^4].

*Note that the tech is still experimental, use it on mainnet at your own risk!*
