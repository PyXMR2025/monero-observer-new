vdo[^1] has released v0.3.3[^2] of the Monero node boostrapper script, *xmr.sh*[^3], with *monero-lws*[^4] and ARM64 support:

> Feedback on Arm64 hardware will be rewarded :)[^5]

### Changes

- Optional pruning
- All images support amr64 architecture
- Monero LWS (Light Wallet Service) daemon (experimental)
- Other minor improvements and fixes

To learn more about the project, consult the project's code repository[^6], the FAQ[^7] section and the previous Monero Observer report[^8].
