larteyoh[^1] is looking to open *a paid bounty program*[^2] for the development of *Neroshop*[^3], an online marketplace for Monero users:

> I am looking for experienced c/c++ programmers to help me out with a project I am currently working on. It is an online marketplace for Monero users. [..] The only prerequisite is that you know how to use cmake.[^4]

### Tasks

```
$1000 - implement peer-to-peer database or network using system APIs
$500 - create a search engine that can search for item names, seller names and products, etc.
$500 - fix and optimize graphics and adding graphical user interface features
$250 - convert and export qr code data to PNG
$100 - fix decoding error in functions using EVP_PKEY_encrypt() and EVP_PKEY_decrypt()
```

Bounties will be paid in XMR for each task.

To learn more and get involved with the project, consult the associated code repository[^3] and contact the maintainer[^5].
