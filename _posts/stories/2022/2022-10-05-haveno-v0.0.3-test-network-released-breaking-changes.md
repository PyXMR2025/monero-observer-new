Haveno[^1] has released version 0.0.3[^2] of its test network with some *breaking changes*:

> [..] be sure to clear your application folder before testing, since there are breaking changes.[^3]

### Changes overview

- ci: remove docs/ from ignored paths in workflows
- process messages on user thread and protocol tasks off user thread
- provide error message if price is unavailable for offer
- fix color scheme of offer book in market view
- fixes based on refactored tests [..]
