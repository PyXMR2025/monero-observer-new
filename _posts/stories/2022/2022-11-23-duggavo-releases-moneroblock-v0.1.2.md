duggavo[^1] has released version 0.1.2[^2] of the trustless XMR block explorer, *MoneroBlock*[^3].

### Changes overview

```
- Fix incorrectly displayed tx size and add --rpc-login
```

To run the software, use the prebuilt binaries[^2] or compile from source. Instructions can be found on Github[^4].

Donate XMR and WOW to the addresses listed in the README[^5] to support MoneroBlock development.
