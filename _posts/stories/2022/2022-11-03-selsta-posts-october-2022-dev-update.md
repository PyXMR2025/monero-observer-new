selsta[^1] has posted a monthly CCS progress report[^2] for October 2022, which includes the following dev updates:

```
Milestone 3:
- Fix fee estimation in GUI, take priority into account
- Fix a bug that would cause the GUI to create log files in unwanted places
- Evaluated the new macOS version is regards to monero compatibility. [..]
- Smaller fixes, improvements and maintenance work (CLI, GUI)
```

Note that *misc* work is not explicitly mentioned in these updates. The full list of changes can be found on Github[^3]'[^4].

This update completes selsta's part-time (Aug-Oct) Monero development CCS proposal[^5].
