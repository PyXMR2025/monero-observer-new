hinto-janaiyo[^1] has released version 0.5.0[^2] of the cross-platform GUI for mining Monero on P2Pool using XMRig, *Gupax*[^3].

### Changes overview

- Added `Simple` P2Pool tab
- Added `Advanced` P2Pool tab
- Added command arguments
- Added fullscreen GUI error handler [..]

The full changelog, sources, SHA256SUM and .asc files can be found on Github[^2] and on the new *gupax.io* website[^4].

According to the latest dev update[^5], we can expect version 1.0.0 to be released *around Christmas time*. 

Consult hinto-janaiyo's CCS proposal[^6] to learn more about the project.
