selsta[^1] has posted a monthly CCS progress report[^2] for September 2022, which includes the following dev updates:

```
Milestone 2:
- Add gitian builds to monero CI
- Prepared release v0.18.1.1 and v0.18.1.2
- Remove unused packages (Qt, ldns) from depends, decrease build time for reproducible builds
- Investigate fee multiplier not being taken into account in fee estimate (GUI)
- Smaller fixes and improvements (CLI, GUI)
```

*Misc* work is not explicitly mentioned in these updates.

A more complete list of changes can be found on Github[^3]'[^4].

Consult the previous Monero Observer report[^5] to learn more about selsta's most recent CCS proposal.
