selsta[^1] has posted a monthly CCS progress report[^2] for November 2022, which includes the following dev updates:

```
Milestone 1:
- Prepare the next update (v0.18.1.3) with Dandelion++ and network stability improvements.
- Various testing in regards to "no incoming peers" bug, fix was found by j-berman.
- Seed node maintenance
- Smaller fixes, improvements and maintenance work (CLI, GUI)
```

Note that *misc* work is not explicitly mentioned in these updates. The full list of changes can be found on Github[^3]'[^4].

This update completes the first milestone of selsta's part-time (Nov-Jan) Monero development CCS proposal[^5].
