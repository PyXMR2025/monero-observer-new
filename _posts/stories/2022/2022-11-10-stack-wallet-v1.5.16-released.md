Stack Wallet[^1] beta version 1.5.16 (build_0088)[^2] has been released with the following bug fixes and improvements:

- Wownero 25 seed restore support
- Allow Firo public or private sends on exchanges
- Address book now sorted alphabetically
- BCH fixes
- Epic notes now should save correctly
- [..]

The release notes for this version, the .APK download and the SHA256 hashes can be found on Github[^2]. 

Consult the previous Monero Observer report[^3] to learn more about the project.
