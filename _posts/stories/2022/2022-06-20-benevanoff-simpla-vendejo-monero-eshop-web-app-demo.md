benevanoff[^1] has shared a demo[^2] for *Simpla Vendejo*[^3], an e-commerce web application for *e-shops that process Monero transactions*:

> For now the app’s feature set is fairly minimalist but if anybody out there starts actually using this and has any feature requests I’ll gladly add it if you open up an issue on the repo. If anybody needs help with setup feel free to PM me.[^4]

### Dependencies

- PHP
- MySQL
- Simpla Vendejo middleware (ExpressJs)[^5]

To learn more about the project, consult the associated Github repositories[^3]'[^5].

To support the development of this software, send tips to the XMR address listed in the *Simpla-Vendejo-PHP* repository's README.md[^6] file.
