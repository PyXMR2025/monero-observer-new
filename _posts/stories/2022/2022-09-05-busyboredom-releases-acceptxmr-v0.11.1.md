busyboredom[^1] has released v0.11.1[^2] of the Monero payment gateway library, *AcceptXMR*[^3]:

> Update monero-rs to 0.18, bringing support for view tags and bulletproofs+.

To learn more about the project, consult the code repository README.md[^4] file and the updated demo[^5] on busyboredom's website.

Note that the library is *young and unproven*, use it at your own risk.
